# Libft
*This activity has been created as part of the 42 curriculum by oaldalgm.*


## Description
Libft is a foundational project in the 42 curriculum.  
Its goal is to recreate a custom C library by re-implementing a collection of essential functions from the standard C library, as well as additional utility functions.    
The final result of this project is a static library file named **libft.a**, which will be reused in manyfuture 42 projects.

## Instructions 
- A standard C compiler like 'cc' or 'gcc' 
- Only allowed functions: 'malloc', 'free', 'write' 
- Code must follow the 42 Norm
